# Udemy Course Listing Generator

Generate the complete Udemy course listing page content for the RxJS Deep Dive course.

Read `readme.md` and `docs/index.md` before generating — all claims must be grounded in the actual course content.

---

Produce all seven sections below in order.

---

### 1 — Course Title (max 60 characters)
Optimised for Udemy search. Include the main keyword "RxJS".

### 2 — Course Subtitle (max 120 characters)
Expand on the title. Include secondary keywords: reactive programming, TypeScript, Observable, operators.

### 3 — Course Description (Udemy HTML format)
Structure:
- **What you'll learn** — 4–6 bullet points, each starting with an action verb
- **Course overview** — 3–4 paragraphs. Hook in the first sentence. Explain who this is for and what makes it different.
- **What's included** — list the 16 modules and 2 appendices as bullet points
- **Requirements** — what the student needs before enrolling
- **Who this course is for** — 3 short bullet points describing the target learner

### 4 — Learning Objectives (shown as checkboxes on Udemy listing)
Write exactly 8 learning objectives. Each must:
- Start with an action verb (Build, Write, Test, Explain, Choose, Design, Debug, Integrate)
- Be specific and measurable
- Be achievable by the end of the course

### 5 — Course Requirements
List 3–5 prerequisites. Be honest — don't over-require or under-require.

### 6 — Target Audience
3 bullet points describing who should take this course. Be specific.

### 7 — SEO Keywords
List 10–15 keywords/phrases a potential student would search for on Udemy to find this course.

---

## Udemy listing quality rules
- No superlatives ("best", "most complete") — Udemy rejects these
- No income/earning claims
- No external URLs in the description body
- All bullet points must be factual and verifiable from the course content
- Description must be at least 200 words
